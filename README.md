# React-Demo
This is a React Demo!

# 下载

````
git clone git@github.com:peter-song/React-Demo.git

cd React-Demo

npm i
````

# 运行

````
npm run dev

open http://localhost:8080

or

npm start

open http://localhost:3000
````
